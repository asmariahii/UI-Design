# Integration web
Bienvenue sur Boutique des Créateurs Indépendants, une application dédiée aux créateurs de vêtements indépendants. Cette plateforme permet à chaque designer de gérer sa propre boutique, d'exposer ses créations et de publier des descriptions détaillées de ses articles.
